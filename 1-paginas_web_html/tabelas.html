Tabelas

As tabelas são listas de dados em duas dimensões e são compostas por linhas e colunas. Portanto, são muito utilizadas para apresentar dados de uma forma organizada.

Elementos de uma tabela

  • table: elemento principal que define a tabela.
  • tr: define uma linha na tabela (table row).
  • th: define uma célula de cabeçalho (table header).
  • td: define uma célula de dados (table data).
