Formulários

O formulário HTML é um formulário de preenchimento de dados ou que resulta em uma ação desejada utilizando a linguagem de marcação HTML. É formado por um ou mais campos. Esses campos podem ser de textos, caixas de seleção, botões, radio buttons e checkboxes utilizando tags do próprio HTML. Dessa forma, o usuário pode interagir com a página e executar ações através desses formulários.

Elementos de um formulário

  • form: elemento principal que define o formulário.
  • input: utilizado para criar campos de entrada de dados, como texto, senha, email, etc.
  • textarea: utilizado para criar áreas de texto maiores, como comentários ou mensagens.
  • button: utilizado para criar botões, como enviar ou resetar o formulário.
  • select: utilizado para criar listas suspensas (dropdowns) com opções pré-definidas.
  • label: utilizado para associar um texto descritivo a um elemento de formulário.
