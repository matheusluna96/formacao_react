Listas

As listas são muito importantes quando queremos listar alguns itens no site e também para a criação de menu de navegação.

Tipos de Listas

  • Listas Ordenadas
  • Listas desordenadas
  • Listas por definição

Listas Ordenadas

As listas ordenadas são aquelas que possuem uma ordem numérica ou alfabética, ou seja, os itens são numerados.

  1. Item 1
  2. Item 2
  3. Item 3

Parâmetro Start

  1. Item 1
  2. Item 2
  3. Item 3

Parâmetro type

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não ordenadas

  • Primeiro item
  • Segundo item
  • Terceiro item

Listas aninhadas

  1. Primeiro item
  2. Segundo item
    1. Item 1
    2. Item 2
    3. Item 3
  3. Terceiro item
